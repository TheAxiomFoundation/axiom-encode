Production external Ed25519 apply-signer (`cmd/axiom-encode-apply-signer`): the protocol v2 signer the supervisor's broker attaches to for a signed, autonomous `encode --apply` CI leg. Ingests one scope-locked key from a pipe/socket descriptor (never env/argv/disk), signs only domain-bound apply payloads, binds to a matching main-branch `workflow_dispatch`/`schedule` run, and ships a launcher that clears the key from its environment before attaching the supervisor.
