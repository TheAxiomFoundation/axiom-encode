Added `find_missing_money_proof_atoms` plus the `proof-validate --require-money-atoms [--money-atoms-only] [--ratchet-file] [--money-atom-root] [--emit-ratchet]` surface, deriving proof obligations for policy-bearing monetary values (currency parameters, currency parameter-table cells, and currency literals in derived formulas) so a repository can no longer ship atom-less money parameters unnoticed. `--money-atoms-only` runs solely the money gate (skipping base proof-tree validation) so a repo-wide CI scan does not re-report unrelated proof issues on unchanged files.
